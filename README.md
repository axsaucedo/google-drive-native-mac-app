## Keyboard Shortcuts

### Go-To Actions

* Go to Drive Home - **CMD + H**
* Go to Default Folder - **CMD + D**
* Set current folder as Default - **CMD + S**

### File actions

* New Doc - **CMD + 1**
* New Slide - **CMD + 2**
* New Sheet - **CMD + 3**
* Close window - **CMD + W**
* Quit App - **CMD + Q**
